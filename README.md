# holdown
ラップトップで右ホールドをするためのツール。ChatGPTで作った。
私のラップトップでは右ホールド（右クリック長押し）に非対応だったので作成した。
二本指日タップで右の長押しが可能であるがその状態でマウスを動かすことはできずまた動か氏終わったあとの正常な右長押しの解除を実装していないので、マウスを使ったほうがいい
## Installation

```
cargo install --path . 
```
もしくはおすすめしないがArchなら

```
makepkg -si
```

## Usage
```
sudo holdown
```
sudo権限がなければevdevに直接アクセスすることができないので必要っぽい。
